# Sketchpad UI — Component Phases

Every phase is a React + Flutter + docs + tests vertical slice.

## Phase 0 — Foundation

Semantic light/dark tokens, six accents, custom accents, hard borders/shadows,
focus, motion, RTL, React provider, Flutter `ThemeExtension`, docs shell.

## Phase 1 — Essentials

Aspect Ratio, Avatar, Badge, Button, Button Group, Card, Item, Kbd, Progress,
Separator, Skeleton, Spinner, Typography.

## Phase 2 — Forms

Calendar, Checkbox, Combobox, Date Picker, Field, Input, Input Group, Input OTP,
Label, Native Select, Radio Group, Select, Slider, Switch, Textarea, Toggle,
Toggle Group.

## Phase 3 — Navigation and layout

Accordion, Breadcrumb, Collapsible, Direction, Navigation Menu, Pagination,
Resizable, Scroll Area, Sidebar, Tabs.

## Phase 4 — Feedback and overlays

Alert, Alert Dialog, Command, Context Menu, Dialog, Drawer, Dropdown Menu,
Hover Card, Menubar, Popover, Sheet, Sonner, Toast, Tooltip.

## Phase 5 — Data and media

Carousel, Chart, Data Table, Empty, Table.

## Phase 6 — Multiplayer messaging

Attachment, Bubble, Marker, Message, Message Scroller.

## Phase 7 — v1.0

Cross-framework API audit, accessibility and golden tests, clean-app
installation tests, npm/pub.dev publication, and versioned documentation.
